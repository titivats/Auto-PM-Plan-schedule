PM Plan Auto Schedule - Portable Package

How to use
1. Copy this folder to another Windows machine.
2. Make sure Microsoft Excel is installed on that machine.
3. Run PMPlanAutoSchedule.exe.
4. Choose the template Excel workbook with the Browse button.
5. Choose the output folder and generate the Jan-Dec files.

Notes
- This package is portable. The EXE can run without the source code folder.
- The app will create app_state.json next to the EXE after it runs.
- If the template path from another machine is no longer valid, just browse to the correct file.
- Excel COM automation is required for file generation.
